# Sales Analytics Platform

End-to-end production data pipeline with Docker, Airflow, dbt, and Power BI.

## Architecture
```
API/CSV → Python ETL → DuckDB → Airflow → dbt → Power BI
```

## Technologies

- **Python**: Data extraction & loading
- **Docker**: Containerization
- **Airflow**: Workflow orchestration
- **dbt**: Data transformations
- **SQL**: Complex queries
- **Power BI**: Dashboards
- **Git**: Version control
